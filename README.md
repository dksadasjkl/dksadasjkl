## 👋 Introduction
안녕하세요, **백엔드 중심의 풀스택 개발자 서창현**입니다.  
서비스의 안정적인 운영을 위해 **설계·개발·배포·운영 전 과정을 고려하는 개발자**를 지향합니다.  
방문해 주셔서 감사합니다.

## 🌟 About Me

<details>
  <summary><strong>나의 일하는 방식 (성장과 책임감)</strong></summary>
  맡은 기능을 끝까지 완성하는 책임감을 가장 중요하게 생각합니다.  
  단순히 동작하는 코드에 그치지 않고, <b>유지보수성과 확장성을 고려한 구조</b>를 고민하며 개발합니다.  
  문제를 해결하는 과정을 성장의 기회로 삼는 개발자입니다.
</details>

<details>
  <summary><strong>나의 소통 방식 (사람과 팀워크)</strong></summary>
  팀원의 의견을 존중하며, 기술적인 판단에는 항상 근거를 바탕으로 소통하려 노력합니다.  
  코드 리뷰와 협업 과정에서 명확한 설명과 합리적인 조율을 통해  
  팀 전체의 개발 효율과 코드 품질을 함께 높이는 역할을 지향합니다.
</details>

<details>
  <summary><strong>학습과 사고 방식 (이해 중심)</strong></summary>
  빠른 구현보다 <b>왜 이렇게 설계해야 하는지</b>를 이해하는 것을 중요하게 생각합니다.  
  공식 문서와 레퍼런스를 기반으로 문제를 분석하고,  
  개인의 성장을 넘어 팀과 함께 성장하는 개발 문화를 지향합니다.
</details>
